CS 4251: Mobile Applications

As part of a team of 4, I built a mobile application that helps users find affordable food choices targetted at college students. The technology stack incorporated MongoDB for the database, Express.js for the back-end, React-Native for the front end, and the Node.js runtime environment.

In addition to coding, the team learned the principles of the agile development cycle. The app was built in sprints throughout the semester. During that time the team also produced business model canvases, drew user stories, and incorporated feedback from each stage towards showcasing an MVP as a live demo at the end of the semester

CS 4641: Machine Learning

As a final project for my machine learning class, I implemented three regression algorithms on a dataset to create an algorithm that predicted house prices.

The three algorithms used were kernel ridge regression, k-neighbors regression and a neural network using the keras library. The initial work used functions in Pandas and Numpy to clean the data and convert/drop categorical data fields. Next, each regresssion algorithm was tuned on a training dataset by adjusting the relevant hyperparameters. Finally the algorithms were tested on a separate "test" dataset with the RMSE value being used to judge accuracy.
